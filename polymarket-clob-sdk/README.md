# polymarket-clob-sdk

SDK for polymarket clob sdk integration.

## Installation

```bash
npm install @jellychain/polymarket-clob-sdk
```

## Usage

```typescript
import { polymarket-clobSDK } from "@jellychain/polymarket-clob-sdk";

const sdk = new polymarket-clobSDK({ /* config */ });
```
