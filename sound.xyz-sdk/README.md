# sound.xyz-sdk

SDK for sound.xyz sdk integration.

## Installation

```bash
npm install @jellychain/sound.xyz-sdk
```

## Usage

```typescript
import { sound.xyzSDK } from "@jellychain/sound.xyz-sdk";

const sdk = new sound.xyzSDK({ /* config */ });
```
