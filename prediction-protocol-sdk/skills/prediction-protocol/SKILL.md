# Prediction Protocol SDK — Agent Skill

Gnosis Conditional Tokens and Augur protocol integration.

## When to Use

- Checking settlement status on Gnosis/Augur markets
- Understanding conditional token mechanics
- Cross-protocol settlement comparison

## Tools

### protocol_get_gnosis_market
Get market data from Gnosis Conditional Tokens Framework.

### protocol_get_augur_market
Get Augur market data.

### protocol_check_settlement
Check if a market is settled and get resolution.

## No API Key Required

Reads on-chain data directly.
