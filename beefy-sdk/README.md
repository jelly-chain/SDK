# beefy-sdk

SDK for beefy sdk integration.

## Installation

```bash
npm install @jellychain/beefy-sdk
```

## Usage

```typescript
import { ueefySDK } from "@jellychain/beefy-sdk";

const sdk = new ueefySDK({ /* config */ });
```
