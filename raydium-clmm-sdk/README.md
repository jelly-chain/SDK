# raydium-clmm-sdk

SDK for raydium clmm sdk integration.

## Installation

```bash
npm install @jellychain/raydium-clmm-sdk
```

## Usage

```typescript
import { raydium-clmmSDK } from "@jellychain/raydium-clmm-sdk";

const sdk = new raydium-clmmSDK({ /* config */ });
```
