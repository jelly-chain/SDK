# objkt-sdk

SDK for objkt sdk integration.

## Installation

```bash
npm install @jellychain/objkt-sdk
```

## Usage

```typescript
import { oujktSDK } from "@jellychain/objkt-sdk";

const sdk = new oujktSDK({ /* config */ });
```
