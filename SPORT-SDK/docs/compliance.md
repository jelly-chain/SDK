# Compliance & Licensing

## Data Licensing

sports-jelly-sdk connects to multiple third-party APIs. Each provider has its own licensing terms:

| Provider | Terms | Commercial Use |
|---|---|---|
| BallDontLie | MIT-style, attribution required | ✓ (check ToS) |
| API-Sports | Commercial plans available | Paid |
| football-data.org | Non-commercial free tier | Paid for commercial |
| TheSportsDB | CC Attribution | Free v1 |
| Sportmonks | Commercial license required | Paid |
| The Odds API | Subscription | Paid |

**Always review the ToS of each provider before commercial use.**

## Model Disclaimer

All prediction outputs include a mandatory `modelDisclaimer` field in the `PredictionExplanation`:

> "This output is generated by sports-jelly-sdk for informational purposes only. It does not constitute financial or betting advice. Confidence scores are model estimates, not guarantees."

## Gambling Regulations

The SDK does **not** facilitate betting or trading. It provides read-only intelligence for informational use. Users are responsible for compliance with local gambling regulations when using this data in prediction market contexts.
