# Cricket SDK — Agent Skill

Cricket intelligence — IPL, ICC events, Big Bash with prediction market integration.

## When to Use

- Getting live cricket scores
- Predicting match outcomes (T20, ODI, Test)
- Analyzing toss impact and pitch conditions

## Tools

### cricket_get_live
Get currently live cricket matches.

### cricket_get_upcoming
Get upcoming matches.

### cricket_predict
Predict match outcome based on team strength, format, venue.

## Required Keys

```bash
CRICAPI_KEY=
```

## Supported Formats

`t20`, `odi`, `test`, `hundred`

## Supported Leagues

`ipl`, `big-bash`, `cpl`, `psl`, `icc-world-cup`, `icc-t20-world-cup`
