# curve-crypto-ng-sdk

SDK for curve crypto ng sdk integration.

## Installation

```bash
npm install @jellychain/curve-crypto-ng-sdk
```

## Usage

```typescript
import { curve-crypto-ngSDK } from "@jellychain/curve-crypto-ng-sdk";

const sdk = new curve-crypto-ngSDK({ /* config */ });
```
