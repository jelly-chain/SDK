# api3-sdk

SDK for api3 sdk integration.

## Installation

```bash
npm install @jellychain/api3-sdk
```

## Usage

```typescript
import { api3SDK } from "@jellychain/api3-sdk";

const sdk = new api3SDK({ /* config */ });
```
