# curve-ren-pool-sdk

SDK for curve ren pool sdk integration.

## Installation

```bash
npm install @jellychain/curve-ren-pool-sdk
```

## Usage

```typescript
import { curve-ren-poolSDK } from "@jellychain/curve-ren-pool-sdk";

const sdk = new curve-ren-poolSDK({ /* config */ });
```
