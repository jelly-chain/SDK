# uniswap-universal-router-sdk

SDK for uniswap universal router sdk integration.

## Installation

```bash
npm install @jellychain/uniswap-universal-router-sdk
```

## Usage

```typescript
import { uniswap-universal-routerSDK } from "@jellychain/uniswap-universal-router-sdk";

const sdk = new uniswap-universal-routerSDK({ /* config */ });
```
