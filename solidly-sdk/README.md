# solidly-sdk

SDK for solidly sdk integration.

## Installation

```bash
npm install @jellychain/solidly-sdk
```

## Usage

```typescript
import { solidlySDK } from "@jellychain/solidly-sdk";

const sdk = new solidlySDK({ /* config */ });
```
