# pancake-nft-sdk

SDK for pancake nft sdk integration.

## Installation

```bash
npm install @jellychain/pancake-nft-sdk
```

## Usage

```typescript
import { pancake-nftSDK } from "@jellychain/pancake-nft-sdk";

const sdk = new pancake-nftSDK({ /* config */ });
```
