# premia-sdk

SDK for premia sdk integration.

## Installation

```bash
npm install @jellychain/premia-sdk
```

## Usage

```typescript
import { premiaSDK } from "@jellychain/premia-sdk";

const sdk = new premiaSDK({ /* config */ });
```
