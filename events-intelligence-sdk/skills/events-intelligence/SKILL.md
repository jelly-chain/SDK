# Events Intelligence SDK — Agent Skill

Event-driven market intelligence — Eventbrite, Ticketmaster data for prediction triggers.

## When to Use

- Finding events that could trigger market movements
- Analyzing concert/festival demand for related tokens
- Detecting sold-out events as bullish signals

## Tools

### events_search
Search events by keyword, city, or category.

### events_analyze_signal
Analyze an event for market signal (bullish/bearish/neutral).

## Required Keys

```bash
EVENTBRITE_TOKEN=
TICKETMASTER_KEY=
```
