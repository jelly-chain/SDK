# Social Sentiment SDK — Agent Skill

Social media sentiment analysis for prediction signals.

## When to Use

- Gauging public sentiment on a team/player/token
- Detecting volume spikes that indicate something is happening
- Finding drama/injury reports from social media

## Tools

### sentiment_analyze
Analyze sentiment for a topic. Returns bullish/bearish/neutral score.

### sentiment_detect_spike
Detect if social volume is spiking for a topic.

### sentiment_detect_drama
Detect drama keywords (trade requests, suspensions, scandals).

## No API Key Required

This SDK analyzes posts you feed it from other sources.
