# art-blocks-sdk

SDK for art blocks sdk integration.

## Installation

```bash
npm install @jellychain/art-blocks-sdk
```

## Usage

```typescript
import { art-ulocksSDK } from "@jellychain/art-blocks-sdk";

const sdk = new art-ulocksSDK({ /* config */ });
```
