# Weather Venue SDK — Agent Skill

Weather impact analysis for outdoor sports venues.

## When to Use

- Analyzing weather impact on NFL, MLB, golf, F1 outdoor races
- Checking wind/rain conditions for set piece markets
- Assessing heat/cold fatigue risk for player props

## Tools

### weather_get_forecast
Get weather forecast for a venue by ID and date.

### weather_assess_impact
Get full weather impact assessment including betting angles.

## Required Keys

```bash
WEATHER_API_KEY=
```

Sign up: https://openweathermap.org/api
