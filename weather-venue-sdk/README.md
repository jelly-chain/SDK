# weather-venue-sdk

SDK for weather venue sdk integration.

## Installation

```bash
npm install @jellychain/weather-venue-sdk
```

## Usage

```typescript
import { weather-venueSDK } from "@jellychain/weather-venue-sdk";

const sdk = new weather-venueSDK({ /* config */ });
```
