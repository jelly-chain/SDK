# element-v2-sdk

SDK for element v2 sdk integration.

## Installation

```bash
npm install @jellychain/element-v2-sdk
```

## Usage

```typescript
import { element-v2SDK } from "@jellychain/element-v2-sdk";

const sdk = new element-v2SDK({ /* config */ });
```
