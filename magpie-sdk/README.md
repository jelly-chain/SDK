# magpie-sdk

SDK for magpie sdk integration.

## Installation

```bash
npm install @jellychain/magpie-sdk
```

## Usage

```typescript
import { magpieSDK } from "@jellychain/magpie-sdk";

const sdk = new magpieSDK({ /* config */ });
```
