# synthetix-perp-sdk

SDK for synthetix perp sdk integration.

## Installation

```bash
npm install @jellychain/synthetix-perp-sdk
```

## Usage

```typescript
import { synthetix-perpSDK } from "@jellychain/synthetix-perp-sdk";

const sdk = new synthetix-perpSDK({ /* config */ });
```
