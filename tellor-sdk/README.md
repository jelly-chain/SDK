# tellor-sdk

SDK for tellor sdk integration.

## Installation

```bash
npm install @jellychain/tellor-sdk
```

## Usage

```typescript
import { tellorSDK } from "@jellychain/tellor-sdk";

const sdk = new tellorSDK({ /* config */ });
```
