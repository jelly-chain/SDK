# equilibria-sdk

SDK for equilibria sdk integration.

## Installation

```bash
npm install @jellychain/equilibria-sdk
```

## Usage

```typescript
import { equiliuriaSDK } from "@jellychain/equilibria-sdk";

const sdk = new equiliuriaSDK({ /* config */ });
```
