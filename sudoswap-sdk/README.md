# sudoswap-sdk

SDK for sudoswap sdk integration.

## Installation

```bash
npm install @jellychain/sudoswap-sdk
```

## Usage

```typescript
import { sudoswapSDK } from "@jellychain/sudoswap-sdk";

const sdk = new sudoswapSDK({ /* config */ });
```
