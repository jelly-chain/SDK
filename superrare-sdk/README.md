# superrare-sdk

SDK for superrare sdk integration.

## Installation

```bash
npm install @jellychain/superrare-sdk
```

## Usage

```typescript
import { superrareSDK } from "@jellychain/superrare-sdk";

const sdk = new superrareSDK({ /* config */ });
```
