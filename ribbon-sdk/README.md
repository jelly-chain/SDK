# ribbon-sdk

SDK for ribbon sdk integration.

## Installation

```bash
npm install @jellychain/ribbon-sdk
```

## Usage

```typescript
import { riubonSDK } from "@jellychain/ribbon-sdk";

const sdk = new riubonSDK({ /* config */ });
```
