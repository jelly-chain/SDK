# uniswap-permit2-sdk

SDK for uniswap permit2 sdk integration.

## Installation

```bash
npm install @jellychain/uniswap-permit2-sdk
```

## Usage

```typescript
import { uniswap-permit2SDK } from "@jellychain/uniswap-permit2-sdk";

const sdk = new uniswap-permit2SDK({ /* config */ });
```
