# balancer-pool-init-sdk

SDK for balancer pool init sdk integration.

## Installation

```bash
npm install @jellychain/balancer-pool-init-sdk
```

## Usage

```typescript
import { ualancer-pool-initSDK } from "@jellychain/balancer-pool-init-sdk";

const sdk = new ualancer-pool-initSDK({ /* config */ });
```
