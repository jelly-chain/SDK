# gamma-sdkit-sdkit

SDK for gamma sdkit sdkit integration.

## Installation

```bash
npm install @jellychain/gamma-sdkit-sdkit
```

## Usage

```typescript
import { gamma-sdkit-sdkitSDK } from "@jellychain/gamma-sdkit-sdkit";

const sdk = new gamma-sdkit-sdkitSDK({ /* config */ });
```
