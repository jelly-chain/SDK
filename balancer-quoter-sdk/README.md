# balancer-quoter-sdk

SDK for balancer quoter sdk integration.

## Installation

```bash
npm install @jellychain/balancer-quoter-sdk
```

## Usage

```typescript
import { ualancer-quoterSDK } from "@jellychain/balancer-quoter-sdk";

const sdk = new ualancer-quoterSDK({ /* config */ });
```
