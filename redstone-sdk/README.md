# redstone-sdk

SDK for redstone sdk integration.

## Installation

```bash
npm install @jellychain/redstone-sdk
```

## Usage

```typescript
import { redstoneSDK } from "@jellychain/redstone-sdk";

const sdk = new redstoneSDK({ /* config */ });
```
