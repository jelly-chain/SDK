# chainlink-sdk

SDK for chainlink sdk integration.

## Installation

```bash
npm install @jellychain/chainlink-sdk
```

## Usage

```typescript
import { chainlinkSDK } from "@jellychain/chainlink-sdk";

const sdk = new chainlinkSDK({ /* config */ });
```
