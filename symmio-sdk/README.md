# symmio-sdk

SDK for symmio sdk integration.

## Installation

```bash
npm install @jellychain/symmio-sdk
```

## Usage

```typescript
import { symmioSDK } from "@jellychain/symmio-sdk";

const sdk = new symmioSDK({ /* config */ });
```
