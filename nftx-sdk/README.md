# nftx-sdk

SDK for nftx sdk integration.

## Installation

```bash
npm install @jellychain/nftx-sdk
```

## Usage

```typescript
import { nftxSDK } from "@jellychain/nftx-sdk";

const sdk = new nftxSDK({ /* config */ });
```
