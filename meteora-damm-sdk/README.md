# meteora-damm-sdk

SDK for meteora damm sdk integration.

## Installation

```bash
npm install @jellychain/meteora-damm-sdk
```

## Usage

```typescript
import { meteora-dammSDK } from "@jellychain/meteora-damm-sdk";

const sdk = new meteora-dammSDK({ /* config */ });
```
