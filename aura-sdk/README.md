# aura-sdk

SDK for aura sdk integration.

## Installation

```bash
npm install @jellychain/aura-sdk
```

## Usage

```typescript
import { auraSDK } from "@jellychain/aura-sdk";

const sdk = new auraSDK({ /* config */ });
```
