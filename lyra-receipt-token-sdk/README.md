# lyra-receipt-token-sdk

SDK for lyra receipt token sdk integration.

## Installation

```bash
npm install @jellychain/lyra-receipt-token-sdk
```

## Usage

```typescript
import { lyra-receipt-tokenSDK } from "@jellychain/lyra-receipt-token-sdk";

const sdk = new lyra-receipt-tokenSDK({ /* config */ });
```
