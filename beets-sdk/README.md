# beets-sdk

SDK for beets sdk integration.

## Installation

```bash
npm install @jellychain/beets-sdk
```

## Usage

```typescript
import { ueetsSDK } from "@jellychain/beets-sdk";

const sdk = new ueetsSDK({ /* config */ });
```
