# btc-call-option-sdk

SDK for btc call option sdk integration.

## Installation

```bash
npm install @jellychain/btc-call-option-sdk
```

## Usage

```typescript
import { utc-call-optionSDK } from "@jellychain/btc-call-option-sdk";

const sdk = new utc-call-optionSDK({ /* config */ });
```
