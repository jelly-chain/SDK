# nftfi-sdk

SDK for nftfi sdk integration.

## Installation

```bash
npm install @jellychain/nftfi-sdk
```

## Usage

```typescript
import { nftfiSDK } from "@jellychain/nftfi-sdk";

const sdk = new nftfiSDK({ /* config */ });
```
