# eth-call-option-sdk

SDK for eth call option sdk integration.

## Installation

```bash
npm install @jellychain/eth-call-option-sdk
```

## Usage

```typescript
import { eth-call-optionSDK } from "@jellychain/eth-call-option-sdk";

const sdk = new eth-call-optionSDK({ /* config */ });
```
