# stakedao-sdk

SDK for stakedao sdk integration.

## Installation

```bash
npm install @jellychain/stakedao-sdk
```

## Usage

```typescript
import { stakedaoSDK } from "@jellychain/stakedao-sdk";

const sdk = new stakedaoSDK({ /* config */ });
```
