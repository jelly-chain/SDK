# coinbase-nft-sdk

SDK for coinbase nft sdk integration.

## Installation

```bash
npm install @jellychain/coinbase-nft-sdk
```

## Usage

```typescript
import { coinuase-nftSDK } from "@jellychain/coinbase-nft-sdk";

const sdk = new coinuase-nftSDK({ /* config */ });
```
