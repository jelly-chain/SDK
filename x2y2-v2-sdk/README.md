# x2y2-v2-sdk

SDK for x2y2 v2 sdk integration.

## Installation

```bash
npm install @jellychain/x2y2-v2-sdk
```

## Usage

```typescript
import { x2y2-v2SDK } from "@jellychain/x2y2-v2-sdk";

const sdk = new x2y2-v2SDK({ /* config */ });
```
