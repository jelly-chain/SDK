# marginfi-sdk

SDK for marginfi sdk integration.

## Installation

```bash
npm install @jellychain/marginfi-sdk
```

## Usage

```typescript
import { marginfiSDK } from "@jellychain/marginfi-sdk";

const sdk = new marginfiSDK({ /* config */ });
```
