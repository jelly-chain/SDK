# balancer-stable-sdk

SDK for balancer stable sdk integration.

## Installation

```bash
npm install @jellychain/balancer-stable-sdk
```

## Usage

```typescript
import { ualancer-stableSDK } from "@jellychain/balancer-stable-sdk";

const sdk = new ualancer-stableSDK({ /* config */ });
```
