# apollox-sdk

SDK for apollox sdk integration.

## Installation

```bash
npm install @jellychain/apollox-sdk
```

## Usage

```typescript
import { apolloxSDK } from "@jellychain/apollox-sdk";

const sdk = new apolloxSDK({ /* config */ });
```
