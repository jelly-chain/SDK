# margin-trade-sdk

SDK for margin trade sdk integration.

## Installation

```bash
npm install @jellychain/margin-trade-sdk
```

## Usage

```typescript
import { margin-tradeSDK } from "@jellychain/margin-trade-sdk";

const sdk = new margin-tradeSDK({ /* config */ });
```
