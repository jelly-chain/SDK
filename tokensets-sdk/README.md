# tokensets-sdk

SDK for tokensets sdk integration.

## Installation

```bash
npm install @jellychain/tokensets-sdk
```

## Usage

```typescript
import { tokensetsSDK } from "@jellychain/tokensets-sdk";

const sdk = new tokensetsSDK({ /* config */ });
```
