# showtime-sdk

SDK for showtime sdk integration.

## Installation

```bash
npm install @jellychain/showtime-sdk
```

## Usage

```typescript
import { showtimeSDK } from "@jellychain/showtime-sdk";

const sdk = new showtimeSDK({ /* config */ });
```
