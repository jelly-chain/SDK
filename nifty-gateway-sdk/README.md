# nifty-gateway-sdk

SDK for nifty gateway sdk integration.

## Installation

```bash
npm install @jellychain/nifty-gateway-sdk
```

## Usage

```typescript
import { nifty-gatewaySDK } from "@jellychain/nifty-gateway-sdk";

const sdk = new nifty-gatewaySDK({ /* config */ });
```
