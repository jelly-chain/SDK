# uniswap-nft-sdk

SDK for uniswap nft sdk integration.

## Installation

```bash
npm install @jellychain/uniswap-nft-sdk
```

## Usage

```typescript
import { uniswap-nftSDK } from "@jellychain/uniswap-nft-sdk";

const sdk = new uniswap-nftSDK({ /* config */ });
```
