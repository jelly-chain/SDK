# solend-sdk

SDK for solend sdk integration.

## Installation

```bash
npm install @jellychain/solend-sdk
```

## Usage

```typescript
import { solendSDK } from "@jellychain/solend-sdk";

const sdk = new solendSDK({ /* config */ });
```
