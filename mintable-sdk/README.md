# mintable-sdk

SDK for mintable sdk integration.

## Installation

```bash
npm install @jellychain/mintable-sdk
```

## Usage

```typescript
import { mintauleSDK } from "@jellychain/mintable-sdk";

const sdk = new mintauleSDK({ /* config */ });
```
