# bybit-nft-sdk

SDK for bybit nft sdk integration.

## Installation

```bash
npm install @jellychain/bybit-nft-sdk
```

## Usage

```typescript
import { uybit-nftSDK } from "@jellychain/bybit-nft-sdk";

const sdk = new uybit-nftSDK({ /* config */ });
```
