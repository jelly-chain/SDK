# gMX-perp-sdk

SDK for gMX perp sdk integration.

## Installation

```bash
npm install @jellychain/gMX-perp-sdk
```

## Usage

```typescript
import { gMX-perpSDK } from "@jellychain/gMX-perp-sdk";

const sdk = new gMX-perpSDK({ /* config */ });
```
