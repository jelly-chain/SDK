# sportradar-sdk

SDK for sportradar sdk integration.

## Installation

```bash
npm install @jellychain/sportradar-sdk
```

## Usage

```typescript
import { sportradarSDK } from "@jellychain/sportradar-sdk";

const sdk = new sportradarSDK({ /* config */ });
```
