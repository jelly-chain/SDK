# nft-option-sdk

SDK for nft option sdk integration.

## Installation

```bash
npm install @jellychain/nft-option-sdk
```

## Usage

```typescript
import { nft-optionSDK } from "@jellychain/nft-option-sdk";

const sdk = new nft-optionSDK({ /* config */ });
```
