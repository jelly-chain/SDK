# long-short-sdk

SDK for long short sdk integration.

## Installation

```bash
npm install @jellychain/long-short-sdk
```

## Usage

```typescript
import { long-shortSDK } from "@jellychain/long-short-sdk";

const sdk = new long-shortSDK({ /* config */ });
```
