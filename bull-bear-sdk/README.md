# bull-bear-sdk

SDK for bull bear sdk integration.

## Installation

```bash
npm install @jellychain/bull-bear-sdk
```

## Usage

```typescript
import { uull-bearSDK } from "@jellychain/bull-bear-sdk";

const sdk = new uull-bearSDK({ /* config */ });
```
