# curve-eth-pool-sdk

SDK for curve eth pool sdk integration.

## Installation

```bash
npm install @jellychain/curve-eth-pool-sdk
```

## Usage

```typescript
import { curve-eth-poolSDK } from "@jellychain/curve-eth-pool-sdk";

const sdk = new curve-eth-poolSDK({ /* config */ });
```
