# bitget-nft-sdk

SDK for bitget nft sdk integration.

## Installation

```bash
npm install @jellychain/bitget-nft-sdk
```

## Usage

```typescript
import { uitget-nftSDK } from "@jellychain/bitget-nft-sdk";

const sdk = new uitget-nftSDK({ /* config */ });
```
