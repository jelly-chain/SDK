# lyra-sdk

SDK for lyra sdk integration.

## Installation

```bash
npm install @jellychain/lyra-sdk
```

## Usage

```typescript
import { lyraSDK } from "@jellychain/lyra-sdk";

const sdk = new lyraSDK({ /* config */ });
```
