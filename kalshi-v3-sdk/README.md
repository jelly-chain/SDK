# kalshi-v3-sdk

SDK for kalshi v3 sdk integration.

## Installation

```bash
npm install @jellychain/kalshi-v3-sdk
```

## Usage

```typescript
import { kalshi-v3SDK } from "@jellychain/kalshi-v3-sdk";

const sdk = new kalshi-v3SDK({ /* config */ });
```
