# Kalshi V3 SDK — Agent Skill

Full Kalshi V3 exchange API — events, markets, orderbook, order placement, portfolio management.

## When to Use

- Browsing Kalshi events and markets
- Getting orderbook data for price analysis
- Placing or cancelling orders
- Managing portfolio positions
- Finding sports/politics/economics markets

## Tools

### kalshi_get_events
Browse events by category (sports, politics, economics, weather).

### kalshi_get_markets
Get markets with current pricing (yes_bid, yes_ask, last_price).

### kalshi_get_orderbook
Get full orderbook depth for a market.

### kalshi_place_order
Place a buy/sell order (requires authentication).

## Required Keys

```bash
KALSHI_API_KEY=
KALSHI_API_SECRET=
```
