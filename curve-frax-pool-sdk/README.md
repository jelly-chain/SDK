# curve-frax-pool-sdk

SDK for curve frax pool sdk integration.

## Installation

```bash
npm install @jellychain/curve-frax-pool-sdk
```

## Usage

```typescript
import { curve-frax-poolSDK } from "@jellychain/curve-frax-pool-sdk";

const sdk = new curve-frax-poolSDK({ /* config */ });
```
