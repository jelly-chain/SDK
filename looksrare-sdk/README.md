# looksrare-sdk

SDK for looksrare sdk integration.

## Installation

```bash
npm install @jellychain/looksrare-sdk
```

## Usage

```typescript
import { looksrareSDK } from "@jellychain/looksrare-sdk";

const sdk = new looksrareSDK({ /* config */ });
```
