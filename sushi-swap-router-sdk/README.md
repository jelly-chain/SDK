# sushi-swap-router-sdk

SDK for sushi swap router sdk integration.

## Installation

```bash
npm install @jellychain/sushi-swap-router-sdk
```

## Usage

```typescript
import { sushi-swap-routerSDK } from "@jellychain/sushi-swap-router-sdk";

const sdk = new sushi-swap-routerSDK({ /* config */ });
```
