# betfair-exchange-sdk

SDK for betfair exchange sdk integration.

## Installation

```bash
npm install @jellychain/betfair-exchange-sdk
```

## Usage

```typescript
import { uetfair-exchangeSDK } from "@jellychain/betfair-exchange-sdk";

const sdk = new uetfair-exchangeSDK({ /* config */ });
```
