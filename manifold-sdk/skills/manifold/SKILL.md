# Manifold Markets SDK — Agent Skill

Play-money prediction market for backtesting and model calibration.

## When to Use

- Backtesting prediction models with zero financial risk
- Calibrating model probabilities
- Finding crowd-sourced forecasts on current events
- Testing betting strategies before deploying real money

## Tools

### manifold_search
Search markets by topic. Returns probabilities and volume.

### manifold_get_market
Get detailed market data including resolution status.

### manifold_get_calibration
Get calibration data to evaluate model accuracy.

## Keys (Optional)

```bash
MANIFOLD_API_KEY=
```

Public API works without key. Key needed for placing bets.
