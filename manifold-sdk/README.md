# manifold-sdk

SDK for manifold sdk integration.

## Installation

```bash
npm install @jellychain/manifold-sdk
```

## Usage

```typescript
import { manifoldSDK } from "@jellychain/manifold-sdk";

const sdk = new manifoldSDK({ /* config */ });
```
