# mexc-nft-sdk

SDK for mexc nft sdk integration.

## Installation

```bash
npm install @jellychain/mexc-nft-sdk
```

## Usage

```typescript
import { mexc-nftSDK } from "@jellychain/mexc-nft-sdk";

const sdk = new mexc-nftSDK({ /* config */ });
```
