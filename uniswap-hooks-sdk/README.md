# uniswap-hooks-sdk

SDK for uniswap hooks sdk integration.

## Installation

```bash
npm install @jellychain/uniswap-hooks-sdk
```

## Usage

```typescript
import { uniswap-hooksSDK } from "@jellychain/uniswap-hooks-sdk";

const sdk = new uniswap-hooksSDK({ /* config */ });
```
