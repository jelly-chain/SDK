# orca-whirlpool-sdk

SDK for orca whirlpool sdk integration.

## Installation

```bash
npm install @jellychain/orca-whirlpool-sdk
```

## Usage

```typescript
import { orca-whirlpoolSDK } from "@jellychain/orca-whirlpool-sdk";

const sdk = new orca-whirlpoolSDK({ /* config */ });
```
