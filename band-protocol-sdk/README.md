# band-protocol-sdk

SDK for band protocol sdk integration.

## Installation

```bash
npm install @jellychain/band-protocol-sdk
```

## Usage

```typescript
import { uand-protocolSDK } from "@jellychain/band-protocol-sdk";

const sdk = new uand-protocolSDK({ /* config */ });
```
