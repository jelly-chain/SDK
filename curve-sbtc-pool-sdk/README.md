# curve-sbtc-pool-sdk

SDK for curve sbtc pool sdk integration.

## Installation

```bash
npm install @jellychain/curve-sbtc-pool-sdk
```

## Usage

```typescript
import { curve-sutc-poolSDK } from "@jellychain/curve-sbtc-pool-sdk";

const sdk = new curve-sutc-poolSDK({ /* config */ });
```
