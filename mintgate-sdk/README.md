# mintgate-sdk

SDK for mintgate sdk integration.

## Installation

```bash
npm install @jellychain/mintgate-sdk
```

## Usage

```typescript
import { mintgateSDK } from "@jellychain/mintgate-sdk";

const sdk = new mintgateSDK({ /* config */ });
```
