# magiceden-sdk

SDK for magiceden sdk integration.

## Installation

```bash
npm install @jellychain/magiceden-sdk
```

## Usage

```typescript
import { magicedenSDK } from "@jellychain/magiceden-sdk";

const sdk = new magicedenSDK({ /* config */ });
```
