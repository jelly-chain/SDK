# political-prediction-sdk

SDK for political prediction sdk integration.

## Installation

```bash
npm install @jellychain/political-prediction-sdk
```

## Usage

```typescript
import { political-predictionSDK } from "@jellychain/political-prediction-sdk";

const sdk = new political-predictionSDK({ /* config */ });
```
