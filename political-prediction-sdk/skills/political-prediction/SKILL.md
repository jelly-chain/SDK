# Political Prediction SDK — Agent Skill

PredictIt integration — elections, politics, policy markets.

## When to Use

- Getting election prediction market prices
- Comparing betting odds to polling data
- Tracking political market sentiment

## Tools

### political_get_presidential
Get current presidential election markets.

### political_search
Search political markets by topic.

### political_compare_polls
Compare market-implied probability to polling average.

## Keys (Optional)

```bash
PREDICTIT_API_KEY=
```

PredictIt API works without key for reading.
