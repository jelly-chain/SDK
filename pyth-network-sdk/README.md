# pyth-network-sdk

SDK for pyth network sdk integration.

## Installation

```bash
npm install @jellychain/pyth-network-sdk
```

## Usage

```typescript
import { pyth-networkSDK } from "@jellychain/pyth-network-sdk";

const sdk = new pyth-networkSDK({ /* config */ });
```
