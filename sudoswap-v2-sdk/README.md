# sudoswap-v2-sdk

SDK for sudoswap v2 sdk integration.

## Installation

```bash
npm install @jellychain/sudoswap-v2-sdk
```

## Usage

```typescript
import { sudoswap-v2SDK } from "@jellychain/sudoswap-v2-sdk";

const sdk = new sudoswap-v2SDK({ /* config */ });
```
