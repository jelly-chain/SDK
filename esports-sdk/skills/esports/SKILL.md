# Esports SDK — Agent Skill

Esports intelligence — LoL, CS2, Dota 2, Valorant with prediction market integration.

## When to Use

- Getting live esports scores and schedules
- Predicting match outcomes based on team ratings
- Finding esports markets on Polymarket

## Tools

### esports_get_live_matches
Get currently live matches across all titles.

### esports_get_upcoming
Get upcoming matches for a title.

### esports_get_tournaments
Get active tournaments.

### esports_predict
Predict match outcome based on team ratings and form.
