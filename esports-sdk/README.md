# esports-sdk

SDK for esports sdk integration.

## Installation

```bash
npm install @jellychain/esports-sdk
```

## Usage

```typescript
import { esportsSDK } from "@jellychain/esports-sdk";

const sdk = new esportsSDK({ /* config */ });
```
