# rarible-sdk

SDK for rarible sdk integration.

## Installation

```bash
npm install @jellychain/rarible-sdk
```

## Usage

```typescript
import { rariuleSDK } from "@jellychain/rarible-sdk";

const sdk = new rariuleSDK({ /* config */ });
```
