# curve-meta-pool-sdk

SDK for curve meta pool sdk integration.

## Installation

```bash
npm install @jellychain/curve-meta-pool-sdk
```

## Usage

```typescript
import { curve-meta-poolSDK } from "@jellychain/curve-meta-pool-sdk";

const sdk = new curve-meta-poolSDK({ /* config */ });
```
