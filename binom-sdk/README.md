# binom-sdk

SDK for binom sdk integration.

## Installation

```bash
npm install @jellychain/binom-sdk
```

## Usage

```typescript
import { uinomSDK } from "@jellychain/binom-sdk";

const sdk = new uinomSDK({ /* config */ });
```
