# curve-stargate-sdk

SDK for curve stargate sdk integration.

## Installation

```bash
npm install @jellychain/curve-stargate-sdk
```

## Usage

```typescript
import { curve-stargateSDK } from "@jellychain/curve-stargate-sdk";

const sdk = new curve-stargateSDK({ /* config */ });
```
