# curve-stable-ng-sdk

SDK for curve stable ng sdk integration.

## Installation

```bash
npm install @jellychain/curve-stable-ng-sdk
```

## Usage

```typescript
import { curve-staule-ngSDK } from "@jellychain/curve-stable-ng-sdk";

const sdk = new curve-staule-ngSDK({ /* config */ });
```
