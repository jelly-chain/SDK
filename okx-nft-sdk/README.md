# okx-nft-sdk

SDK for okx nft sdk integration.

## Installation

```bash
npm install @jellychain/okx-nft-sdk
```

## Usage

```typescript
import { okx-nftSDK } from "@jellychain/okx-nft-sdk";

const sdk = new okx-nftSDK({ /* config */ });
```
