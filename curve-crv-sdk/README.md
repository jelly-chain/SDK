# curve-crv-sdk

SDK for curve crv sdk integration.

## Installation

```bash
npm install @jellychain/curve-crv-sdk
```

## Usage

```typescript
import { curve-crvSDK } from "@jellychain/curve-crv-sdk";

const sdk = new curve-crvSDK({ /* config */ });
```
