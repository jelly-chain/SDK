# Line Movement SDK — Agent Skill

Historical odds tracking and line movement analysis.

## When to Use

- Tracking how odds change over time for a fixture
- Detecting sharp money and steam moves
- Finding value bets where model disagrees with line

## Tools

### line_get_movement
Get full line movement history for a fixture.

### line_detect_steam
Detect rapid line movements indicating sharp money.

### line_analyze_value
Compare model probability to current line for edge detection.

## No API Key Required

This SDK tracks odds from data you feed it.
