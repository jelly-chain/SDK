# teia-sdk

SDK for teia sdk integration.

## Installation

```bash
npm install @jellychain/teia-sdk
```

## Usage

```typescript
import { teiaSDK } from "@jellychain/teia-sdk";

const sdk = new teiaSDK({ /* config */ });
```
