# pendle-sdk

SDK for pendle sdk integration.

## Installation

```bash
npm install @jellychain/pendle-sdk
```

## Usage

```typescript
import { pendleSDK } from "@jellychain/pendle-sdk";

const sdk = new pendleSDK({ /* config */ });
```
