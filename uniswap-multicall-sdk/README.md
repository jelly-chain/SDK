# uniswap-multicall-sdk

SDK for uniswap multicall sdk integration.

## Installation

```bash
npm install @jellychain/uniswap-multicall-sdk
```

## Usage

```typescript
import { uniswap-multicallSDK } from "@jellychain/uniswap-multicall-sdk";

const sdk = new uniswap-multicallSDK({ /* config */ });
```
