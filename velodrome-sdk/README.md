# velodrome-sdk

SDK for velodrome sdk integration.

## Installation

```bash
npm install @jellychain/velodrome-sdk
```

## Usage

```typescript
import { velodromeSDK } from "@jellychain/velodrome-sdk";

const sdk = new velodromeSDK({ /* config */ });
```
