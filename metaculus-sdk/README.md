# metaculus-sdk

SDK for metaculus sdk integration.

## Installation

```bash
npm install @jellychain/metaculus-sdk
```

## Usage

```typescript
import { metaculusSDK } from "@jellychain/metaculus-sdk";

const sdk = new metaculusSDK({ /* config */ });
```
