# pendle-sy-sdk

SDK for pendle sy sdk integration.

## Installation

```bash
npm install @jellychain/pendle-sy-sdk
```

## Usage

```typescript
import { pendle-sySDK } from "@jellychain/pendle-sy-sdk";

const sdk = new pendle-sySDK({ /* config */ });
```
