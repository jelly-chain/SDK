# curve-tricrypto-sdk

SDK for curve tricrypto sdk integration.

## Installation

```bash
npm install @jellychain/curve-tricrypto-sdk
```

## Usage

```typescript
import { curve-tricryptoSDK } from "@jellychain/curve-tricrypto-sdk";

const sdk = new curve-tricryptoSDK({ /* config */ });
```
