# symmetry-sdk

SDK for symmetry sdk integration.

## Installation

```bash
npm install @jellychain/symmetry-sdk
```

## Usage

```typescript
import { symmetrySDK } from "@jellychain/symmetry-sdk";

const sdk = new symmetrySDK({ /* config */ });
```
