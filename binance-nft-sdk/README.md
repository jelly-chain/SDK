# binance-nft-sdk

SDK for binance nft sdk integration.

## Installation

```bash
npm install @jellychain/binance-nft-sdk
```

## Usage

```typescript
import { uinance-nftSDK } from "@jellychain/binance-nft-sdk";

const sdk = new uinance-nftSDK({ /* config */ });
```
