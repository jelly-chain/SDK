# espn-live-sdk

SDK for espn live sdk integration.

## Installation

```bash
npm install @jellychain/espn-live-sdk
```

## Usage

```typescript
import { espn-liveSDK } from "@jellychain/espn-live-sdk";

const sdk = new espn-liveSDK({ /* config */ });
```
