# ESPN Live SDK — Agent Skill

Free, no-API-key sports data — real-time scores, standings, schedules.

## When to Use

- Need quick sports scores without API key setup
- Fallback when other sports APIs are rate-limited
- Getting standings and schedules for major leagues

## Tools

### espn_get_scores
Get live/final scores for a league.

### espn_get_standings
Get league standings.

### espn_get_schedule
Get schedule for a specific date.

## No API Key Required

ESPN public API is free and unlimited.

## Supported Leagues

`nfl`, `nba`, `mlb`, `nhl`, `mls`, `epl`, `ucl`, `ncaaf`, `ncaab`
