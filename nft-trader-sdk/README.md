# nft-trader-sdk

SDK for nft trader sdk integration.

## Installation

```bash
npm install @jellychain/nft-trader-sdk
```

## Usage

```typescript
import { nft-traderSDK } from "@jellychain/nft-trader-sdk";

const sdk = new nft-traderSDK({ /* config */ });
```
