# fxhash-sdk

SDK for fxhash sdk integration.

## Installation

```bash
npm install @jellychain/fxhash-sdk
```

## Usage

```typescript
import { fxhashSDK } from "@jellychain/fxhash-sdk";

const sdk = new fxhashSDK({ /* config */ });
```
