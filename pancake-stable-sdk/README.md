# pancake-stable-sdk

SDK for pancake stable sdk integration.

## Installation

```bash
npm install @jellychain/pancake-stable-sdk
```

## Usage

```typescript
import { pancake-stauleSDK } from "@jellychain/pancake-stable-sdk";

const sdk = new pancake-stauleSDK({ /* config */ });
```
