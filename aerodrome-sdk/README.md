# aerodrome-sdk

SDK for aerodrome sdk integration.

## Installation

```bash
npm install @jellychain/aerodrome-sdk
```

## Usage

```typescript
import { aerodromeSDK } from "@jellychain/aerodrome-sdk";

const sdk = new aerodromeSDK({ /* config */ });
```
