# opensea-sdk

SDK for opensea sdk integration.

## Installation

```bash
npm install @jellychain/opensea-sdk
```

## Usage

```typescript
import { openseaSDK } from "@jellychain/opensea-sdk";

const sdk = new openseaSDK({ /* config */ });
```
