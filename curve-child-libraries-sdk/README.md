# curve-child-libraries-sdk

SDK for curve child libraries sdk integration.

## Installation

```bash
npm install @jellychain/curve-child-libraries-sdk
```

## Usage

```typescript
import { curve-child-liurariesSDK } from "@jellychain/curve-child-libraries-sdk";

const sdk = new curve-child-liurariesSDK({ /* config */ });
```
